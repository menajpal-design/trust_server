# 🇧🇩 trust_server - UnionDesk Backend API

Enterprise Production-Ready Backend API for UnionDesk (TRUST) SaaS Organization Management Platform.

---

## 🚀 Features

- **Bangladesh 9-Tier Geo Hierarchy**: Division, District, Upazila, Municipality, Union, Ward, Village, Mohalla, Custom Area.
- **13-Tier Committee Hierarchy**: National, Division, District, Upazila, Union, Ward, Village, School, College, Mosque, Market, Women's, Youth committees.
- **Finance & Cashbook**: Ledger entries, period closing, PDF receipts, WhatsApp sharing, member dues, and donor campaigns.
- **Events & QR Tickets**: Event scheduling, ticket generation, QR ticket verification, and attendance tracking.
- **Digital Notice Board, E-Voting & Document Vault**: Notice broadcast, meeting resolutions, secret ballot polls, and PDF document storage.
- **Mailtrap Integration**: Transactional welcome credentials emails & password reset.
- **MongoDB Atlas Connection Caching**: Optimized for Vercel Serverless Function deployments.

---

## 🛠️ Tech Stack

- Node.js, Express.js, MongoDB (Mongoose), JWT, Winston, Zod, ExcelJS, PDFKit, QRCode, Mailtrap.
